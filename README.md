# Beauty Demo Bot

Демо Telegram-бота для записи в салон красоты.

## Что внутри

- приветствие и главное меню;
- примеры работ по кнопке;
- выбор услуги, даты и времени;
- запрос имени клиента;
- запрос номера через кнопку Telegram;
- просмотр активной записи через кнопку "Моя запись";
- отмена записи с обязательным вопросом о причине;
- напоминание примерно за 24 часа до визита;
- сохранение заявок и отмен в `appointments.json`.

## Локальный запуск

1. Создайте бота через `@BotFather` и получите токен.
2. В PowerShell запустите:

```powershell
$env:TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН_ОТ_BOTFATHER"
node bot.mjs
```

3. Откройте бота в Telegram и нажмите `/start`.

## Деплой на Koyeb

На Koyeb этот бот нужно запускать как Worker, потому что он не открывает сайт, а постоянно слушает Telegram через polling.

1. Загрузите эту папку в отдельный GitHub-репозиторий.
2. Откройте Koyeb и создайте новый сервис.
3. Выберите GitHub-репозиторий с ботом.
4. Тип сервиса выберите `Worker`.
5. Build method: Dockerfile.
6. Start command можно оставить пустым, Dockerfile уже запускает `npm start`.
7. В Environment variables добавьте:

```text
TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
```

8. Нажмите Deploy.

Важно: `appointments.json` хранится в файловой системе сервиса. Для демо этого достаточно, но для реального салона лучше подключить базу данных или Google Sheets, чтобы записи не терялись после перезапуска сервиса.
